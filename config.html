👋 Avant de commencer
La configuration se fait en copiant un bloc de texte technique depuis un autre site (Firebase). C'est beaucoup plus simple avec un clavier d'ordinateur qu'au doigt sur mobile.
Si possible, ouvrez ce lien sur un ordinateur, vous pourrez ensuite scanner un QR code avec ce téléphone.
🔧 Connecter votre base de données
Chaque famille possède sa propre base de données privée (Firebase), gratuite, sur laquelle personne d'autre n'a accès. Suivez ces étapes une seule fois.
1
Allez sur console.firebase.google.com et connectez-vous avec un compte Google.
2
Cliquez sur « Ajouter un projet », donnez-lui un nom (ex : votre prénom + Bébé), puis créez-le.
3
Dans le menu de gauche, ouvrez « Firestore Database » → « Créer une base de données ». Choisissez une région en Europe et le mode Natif.
4
Dans l'onglet « Règles », activez le mode test (accès ouvert 30 jours, à reconduire ensuite).
5
Retournez sur ⚙️ « Paramètres du projet » → en bas, cliquez sur l'icône Web « </> » pour enregistrer une application, donnez-lui un nom.
6
Firebase affiche un bloc de code contenant firebaseConfig = {"{"} ... {"}"}. Copiez tout ce bloc entre accolades et collez-le ci-dessous.
✓ Connexion réussie
Scannez ce code avec le téléphone de chaque parent : il transmet à la fois la connexion à la base de données et le prénom/date de naissance du bébé, en une seule fois. Vous pourrez toujours retrouver la configuration Firebase dans la console Firebase (Paramètres du projet → Vos applications) si besoin.
